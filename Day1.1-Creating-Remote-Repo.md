## Phase 1: Creating the Remote Repository on GitHub

The **remote repository** is where your project code lives on GitHub's servers.

### 1. Access GitHub

* Go to [github.com](https://github.com).
* **Sign up** for a new account or **log in** if you are an existing user.

### 2. Start a New Repository

1.  After logging in, click the **+** icon in the upper-right corner of the page.
2.  Select **New repository** from the dropdown menu.

### 3. Define Repository Details

Fill out the form with the following details:

| Field | Description | Example |
| :--- | :--- | :--- |
| **Owner** | This should be your GitHub username. | `user_name` |
| **Repository name** | A short, unique, and memorable name for your project. | `resume` |
| **Description** (Optional) | Briefly explain the purpose of the project. | *My Digital Resume website files.* |
| **Visibility** | * **Public:** Visible to everyone. Recommended for open-source. <br> * **Private:** Only visible to you and selected collaborators. | *Public* |
| **Initialize this repository with** | Check **Add a README file** to immediately create a file that describes your project. | $\checkmark$ *Add a README file* |

### 4. Finalize Creation

* Click the green **Create repository** button at the bottom of the page.
